# Paradise Nursery - E-Plant Shopping Application

Paradise Nursery is a **React-based web application** designed for plant enthusiasts to explore and purchase a wide variety of plants. The application provides a clean, responsive UI with a seamless shopping experience — from browsing categories to managing a shopping cart.

---

## Features

### Interactive Landing Page
- Welcoming entry screen with an **"About Us"** section  
- Clear **call-to-action** to start shopping  

### Categorized Product Listing
- Plants organized into meaningful categories:
  - Air Purifying Plants  
  - Aromatic Fragrant Plants  
  - Insect Repellent Plants  
  - Medicinal Plants  
  - Low Maintenance Plants  

### State Management with Redux
- Built using **Redux Toolkit**
- Ensures centralized and consistent cart state across components  

## 🛠️ Tech Stack

| Technology        | Usage                          |
|------------------|--------------------------------|
| **Frontend**     | React.js                       |
| **State Management** | Redux Toolkit, React-Redux |
| **Build Tool**   | Vite                           |
| **Styling**      | CSS3 (Flexbox + Media Queries) |
